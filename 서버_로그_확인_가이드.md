# 🔍 서버 로그 확인 가이드

## 📍 서버 콘솔 찾기

개발 서버가 실행 중인 터미널 창을 찾으세요.

### VS Code에서 확인

1. VS Code 하단 패널의 **"Terminal"** 탭 클릭
2. 또는 **"Output"** 탭에서 로그 확인

### PowerShell/CMD에서 확인

개발 서버를 실행한 터미널 창 확인

---

## 🔍 확인할 로그

"AI로 방송 만들기" 버튼을 클릭한 후 다음 로그를 확인하세요:

### 1. API 요청 수신 로그

```
[API] 받은 요청 데이터:
  keyword: 브레이크타임
  mood: 정중하게
  bgmUrl: https://firebasestorage.googleapis.com/...
  bgmUrl 타입: string
  bgmUrl 길이: 250
```

### 2. BGM 믹싱 시작 로그

```
[API] BGM 믹싱 시작 전:
  voiceBuffer 길이: 45678 bytes
  bgmUrl: https://...

[BGM 믹싱] 시작: BGM URL=https://...
```

### 3. BGM 다운로드 로그

**성공 시:**
```
[BGM 믹싱] 2단계: BGM 다운로드 중...
[BGM 믹싱] ✅ BGM 다운로드 완료: 123456 bytes
```

**실패 시:**
```
[BGM 믹싱] ❌ BGM 다운로드 실패: HTTP 403
[BGM 믹싱] 응답 내용: ...
```

### 4. FFmpeg 실행 로그

**성공 시:**
```
[BGM 믹싱] 4단계: 오디오 믹싱 시작...
[BGM 믹싱] FFmpeg 실행 시작
[BGM 믹싱] 진행률: 50%
[BGM 믹싱] ✅ FFmpeg 처리 완료
```

**실패 시:**
```
[BGM 믹싱] ❌ FFmpeg 에러 발생
[BGM 믹싱] 에러 메시지: ...
```

### 5. 최종 결과 로그

```
[API] BGM 믹싱 완료:
  finalBuffer 길이: 567890 bytes
  BGM 믹싱 여부: 예
```

## 📝 로그 복사 방법

1. 서버 콘솔에서 관련 로그 선택
2. 마우스 우클릭 → "복사"
3. 또는 전체 콘솔 내용을 스크린샷으로 공유

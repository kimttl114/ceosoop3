# 📶 매장 와이파이 QR 코드 생성기 기획서

## 🎯 핵심 가치
**"고객이 '와이파이 비번 뭐예요?' 라고 안 물어보게!"**

---

## 💡 기능 개요

### 문제점
- 매일 손님이 와이파이 비밀번호 물어봄
- 알바생마다 답변이 다름
- 손님이 비번 입력 실패 → 다시 물어봄
- 수기로 쓴 포스터는 지저분함

### 해결책
**QR 코드 한 번 찍으면 자동 연결!**
- 예쁜 포스터 자동 생성
- A4로 프린트해서 매장에 붙이기
- 손님이 카메라로 찍으면 자동 연결


## 🛠️ 기술 구현

### QR 코드 생성 포맷
```
WIFI:T:WPA;S:CafeDeCEO_Guest;P:welcome2024!;;
```

- **T**: 암호화 타입 (WPA, WEP, nopass)
- **S**: SSID (네트워크 이름)
- **P**: 비밀번호
- **H**: 숨김 여부 (true/false)

### 사용 라이브러리
- `qrcode.react` - QR 코드 생성
- `html2canvas` - HTML을 이미지로 변환
- `jsPDF` - PDF 생성
- `react-to-print` - 프린트 기능

```
1. 사장님이 앱 접속

```

### 시나리오 2: 치킨집 알바생용

```
5. 손님: 카메라로 찍음 → 자동 연결
6. ✅ 알바생이 비번 외우지 않아도 됨!

```

```
2. 앱에서 새 비번 입력
3. 새 QR 코드 생성 (1분 소요)
4. 프린트해서 교체
5. ✅ 기존 손님들도 새 QR로 쉽게 재연결!
```

## 📐 프린트 사양

### A4 포스터 (추천)
- **크기**: 210mm x 297mm
- **해상도**: 300 DPI
- **QR 크기**: 100mm x 100mm
- **여백**: 상하좌우 20mm
- **용도**: 벽 부착용, 테이블용

### A5 카드
- **크기**: 148mm x 210mm
- **해상도**: 300 DPI
- **QR 크기**: 70mm x 70mm
- **용도**: 테이블용, 메뉴판 삽입

### 엽서 크기
- **크기**: 100mm x 148mm
- **해상도**: 300 DPI
- **QR 크기**: 50mm x 50mm
- **용도**: 스탠드형, 카드형

## 🎨 UI/UX 고려사항

### 1. 입력 편의성
- 비밀번호 보기/숨기기 토글
- 실시간 QR 미리보기
- 오타 방지 (SSID 대소문자 구분 경고)

### 2. 시각적 피드백
- QR 코드 스캔 가능 여부 체크
- 색상 대비 확인 (QR이 잘 보이는지)
- 프린트 미리보기

### 3. 에러 처리
- SSID 빈칸 경고
- 비밀번호 특수문자 이스케이프
- 너무 긴 텍스트 경고

### 4. 모바일 최적화
- 터치 친화적 버튼
- 세로 모드 최적화
- 프린트는 데스크톱 권장 안내

## 🚀 차별화 포인트

### vs 일반 QR 생성기
| 항목 | 일반 생성기 | 우리 앱 |
|------|------------|---------|
| 와이파이 전용 | ❌ | ✅ |
| 예쁜 템플릿 | ❌ | ✅ 20종+ |
| 프린트 최적화 | ❌ | ✅ A4, A5 |
| 업종별 디자인 | ❌ | ✅ 8종 |
| 한글 지원 | ⚠️ 부족 | ✅ 완벽 |
| 사장님 맞춤 | ❌ | ✅ 100% |

### 경쟁 우위
1. **자영업자 특화**: 매장 분위기에 맞는 디자인
2. **한 번에 완성**: 정보 입력 → 프린트 → 부착
3. **무료**: 광고 없음, 워터마크 없음
4. **쉬움**: IT 몰라도 3분이면 완성

## 📊 기대 효과

### 사장님 입장
- ⏰ 시간 절약: 비번 알려주는 시간 0초
- 💰 비용 절약: 디자인 외주 불필요
- 😊 스트레스 감소: 같은 질문 안 들음

- ✅ 편리함: QR 찍으면 끝
- ⚡ 빠름: 타이핑 불필요
- 🎯 정확함: 오타 걱정 없음

### 알바생 입장
- 📝 암기 불필요: 비번 외우지 않아도 됨
- 😌 업무 간소화: "저기 QR 찍으세요" 끝
- 🆕 신규 알바도 OK: 교육 시간 단축

====================================
초보자를 위한 배포 가이드 (한글)
====================================

📌 가장 쉬운 방법: GitHub Desktop 사용

1단계: GitHub Desktop 설치
---------------------------
① https://desktop.github.com 접속
② "Download for Windows" 클릭
③ 설치 파일 실행 후 기본 설정으로 설치


2단계: GitHub에 코드 업로드
---------------------------
① GitHub Desktop 실행
② GitHub 계정으로 로그인 (없으면 가입)
③ File → Add Local Repository
④ 프로젝트 폴더 선택
   (C:\Users\user\Desktop\자영업자 블라인드)
⑤ 왼쪽 하단 "Publish repository" 클릭
⑥ 저장소 이름: ceo-blind
⑦ "Publish repository" 클릭


3단계: Vercel에서 배포
----------------------
① https://vercel.com 접속
② "Sign Up" → "Continue with GitHub" 클릭
③ "Add New Project" 클릭
④ ceo-blind 저장소 선택 → "Import" 클릭
⑤ 환경 변수 추가:
   - OPENAI_API_KEY = (본인의 OpenAI API 키)
   - NTS_API_KEY = (본인의 국세청 API 키)
⑥ "Deploy" 클릭
⑦ 2-3분 대기


4단계: 완료!
-----------
배포가 완료되면 URL이 생성됩니다!
예: https://ceo-blind.vercel.app

이 URL로 접속하면 웹사이트가 열립니다!


====================================
문제 해결
====================================

Q: Git이 설치되어 있지 않다면?
A: https://git-scm.com/download/win 에서 다운로드

Q: GitHub 계정이 없다면?
A: https://github.com 에서 무료 가입

Q: API 키가 없다면?
A: 
- OpenAI: https://platform.openai.com/api-keys
- 국세청: https://www.data.go.kr (공공데이터포털)

Q: 배포 중 오류가 발생하면?
A: Vercel 대시보드의 "Deployments" 탭에서 로그 확인

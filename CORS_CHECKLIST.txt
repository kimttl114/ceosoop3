CORS 설정 확인 체크리스트
==========================================

✅ Google Cloud Console 접속 확인
   https://console.cloud.google.com

✅ Storage Bucket 찾기
   Cloud Storage > Buckets > ceo-blaind.firebasestorage.app

✅ CORS 설정 확인
   Configuration 탭 > CORS 섹션

✅ 필수 항목 확인
   ☐ "OPTIONS" 메서드 포함
   ☐ "http://localhost:3000" origin 포함
   ☐ 모든 필수 헤더 포함

✅ 설정 저장
   ☐ Save 버튼 클릭
   ☐ 1-2분 대기

✅ 브라우저 재시작
   ☐ 모든 창 닫기
   ☐ 브라우저 다시 열기

✅ 테스트
   ☐ localhost:3000 접속
   ☐ 이미지 업로드 시도
   ☐ Network 탭에서 OPTIONS 요청이 200 OK인지 확인

==========================================

가장 중요한 것: "OPTIONS" 메서드가 반드시 포함되어야 합니다!

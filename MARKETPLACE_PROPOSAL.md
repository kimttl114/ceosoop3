# 중고장터 기능 페이지 제안서

## 📋 개요

자영업자 커뮤니티에 특화된 중고장터 기능을 별도 페이지로 구현하여, 자영업자들이 사용하는 장비, 기기, 가구 등을 중고로 거래할 수 있는 마켓플레이스를 제공합니다.

---

## 🎯 핵심 가치 제안

### 1. 자영업자 특화 카테고리
- **음식점 장비**: 주방기기, 포스기, 냉장고, 오븐 등
- **카페 장비**: 에스프레소 머신, 그라인더, 케이크 쇼케이스 등
- **가구/인테리어**: 테이블, 의자, 카운터, 조명 등
- **사무용품**: 계산기, 프린터, 복사기 등
- **기타**: 간판, 배너, 포장재 등

### 2. 지역 기반 매칭
- 사용자 설정 지역 기반 상품 우선 표시
- 거래 가능 지역 필터링
- 직거래/배송 옵션 선택

### 3. 신뢰도 시스템
- 사업자 인증 사용자 우선 표시
- 거래 후기 및 평점 시스템
- 안전 거래 가이드 제공

## 🏗️ 기능 설계

### 페이지 구조: `/marketplace`

#### 1. 메인 페이지 (`/marketplace`)
**레이아웃:**
```
┌─────────────────────────────────────┐
│  헤더: 검색바 + 필터 + 글쓰기 버튼   │
├─────────────────────────────────────┤
│  카테고리 탭 (음식점/카페/가구/기타) │
├─────────────────────────────────────┤
│  필터: 지역 | 가격 | 상태 | 거래방식 │
├─────────────────────────────────────┤
│  상품 그리드 (카드 형태)              │
│  [이미지] [제목] [가격] [지역]       │
│  [이미지] [제목] [가격] [지역]       │
└─────────────────────────────────────┘
```

**주요 기능:**
- 상품 검색 (제목, 설명, 태그)
- 카테고리별 필터링
- 지역별 필터링
- 가격대 필터링 (슬라이더)
- 거래 방식 필터 (직거래/배송/협의)
- 상태 필터 (새상품/중고/하자있음)
- 정렬 옵션 (최신순/가격낮은순/가격높은순/인기순)

#### 2. 상품 상세 페이지 (`/marketplace/[id]`)
**레이아웃:**
```
┌─────────────────────────────────────┐
│  이미지 갤러리 (스와이프)             │
├─────────────────────────────────────┤
│  제목                                │
│  가격 | 상태 | 지역                  │
├─────────────────────────────────────┤
│  판매자 정보                         │
│  [프로필] 이름 | 인증뱃지 | 평점    │
│  [쪽지보내기] [프로필보기]           │
├─────────────────────────────────────┤
│  상품 설명                           │
│  상세 정보 (카테고리, 구매시기 등)   │
├─────────────────────────────────────┤
│  거래 정보                           │
│  거래방식 | 거래가능지역             │
├─────────────────────────────────────┤
│  [관심상품] [쪽지보내기] [신고하기]  │
└─────────────────────────────────────┘
```

**주요 기능:**
- 이미지 갤러리 (최대 10장)
- 판매자 프로필 및 거래 이력
- 관심상품 등록/해제
- 쪽지 보내기 (거래 문의)
- 신고 기능
- 판매자 다른 상품 보기

#### 3. 상품 등록 페이지 (`/marketplace/create`)
**폼 구성:**
- 제목 (필수)
- 카테고리 선택 (필수)
- 가격 (필수, 숫자만)
- 상태 선택 (새상품/중고/하자있음)
- 상품 설명 (필수, 최소 20자)
- 이미지 업로드 (최소 1장, 최대 10장)
- 거래 방식 (직거래/배송/협의)
- 거래 가능 지역 (지역 선택)
- 연락 방법 (쪽지/전화번호 선택)

**유효성 검사:**
- 제목: 2-50자
- 가격: 1원 이상
- 이미지: 최소 1장 필수
- 설명: 20자 이상

#### 4. 내 상품 관리 (`/marketplace/my`)
**탭 구성:**
- 판매중 (기본)
- 예약중
- 판매완료
- 관심상품

**기능:**
- 상품 수정/삭제
- 판매 상태 변경 (판매중 → 예약중 → 판매완료)
- 조회수 확인
- 관심 등록 수 확인

## 💾 데이터베이스 구조

### Firestore 컬렉션: `marketplace`

```typescript
interface MarketplaceItem {
  id: string
  title: string                    // 제목
  description: string              // 설명
  category: string                 // 카테고리 (음식점/카페/가구/기타)
  subCategory?: string             // 세부 카테고리
  price: number                    // 가격
  condition: 'new' | 'used' | 'damaged'  // 상태
  images: string[]                 // 이미지 URL 배열
  sellerId: string                 // 판매자 UID
  sellerName: string               // 판매자 이름
  sellerRegion: string              // 판매자 지역
  tradeMethod: 'direct' | 'delivery' | 'negotiable'  // 거래방식
  tradeRegion: string[]             // 거래 가능 지역 배열
  contactMethod: 'message' | 'phone'  // 연락방법
  phoneNumber?: string             // 전화번호 (선택)
  status: 'selling' | 'reserved' | 'sold'  // 판매상태
  views: number                    // 조회수
  likes: number                    // 관심 등록 수
  createdAt: Timestamp             // 등록일
  updatedAt: Timestamp             // 수정일
  soldAt?: Timestamp               // 판매완료일
  tags?: string[]                  // 태그 (검색용)
}
```

### 서브컬렉션: `marketplace/{itemId}/likes`
```typescript
interface Like {
  userId: string
  createdAt: Timestamp
}
```

### 서브컬렉션: `marketplace/{itemId}/inquiries`
```typescript
interface Inquiry {
  userId: string
  userName: string
  message: string
  createdAt: Timestamp
  read: boolean
}
```

## 🚀 구현 우선순위

### Phase 1 (MVP)
1. ✅ 상품 등록/수정/삭제
2. ✅ 상품 목록 조회 (카테고리, 지역 필터)
3. ✅ 상품 상세 페이지
4. ✅ 관심상품 등록/해제
5. ✅ 쪽지 보내기 (거래 문의)

### Phase 2 (고도화)
1. 검색 기능 강화 (태그, 자동완성)
2. 판매자 프로필 및 거래 이력
3. 거래 후기 시스템
4. 알림 기능 (관심상품 가격 변동, 새 댓글 등)

### Phase 3 (프리미엄)
1. 상품 추천 시스템 (AI 기반)
2. 가격 비교 기능
3. 경매 기능
4. 중고가 자동 평가 (AI)

- 게시판, 쪽지, 인증 시스템과 연계
